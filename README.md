
# GameOff2025 Unity Project

Welcome to the GameOff2025 Unity project. This repository provides a clean starting point for your team, with Photon PUN2 pre-installed for seamless multiplayer development.

## Project Overview

- **Engine:** Unity (WebGL compatible)
- **Networking:** Photon PUN2 (imported and ready to use)
- **Status:** Blank project—no custom scripts, scenes, or assets

## Getting Started

1. Clone or download this repository.
2. Open the project in Unity Hub.
3. Ensure the Photon PUN2 asset is present in `Assets/Photon`.
4. Begin developing your game features, scenes, and multiplayer logic.

## Multiplayer Development

- Photon PUN2 is set up for easy integration of real-time multiplayer features.
- Add your own scripts, prefabs, and scenes to build your game.
- Refer to the [Photon PUN2 documentation](https://doc.photonengine.com/en-us/pun/current/getting-started/pun-intro) for networking guidance.

## Contribution Guidelines

- Use feature branches for new gameplay systems or assets.
- Submit pull requests for code review and integration.
- Keep the main branch stable and free of compilation errors.
