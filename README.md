# GameOff2025 Unity Project

This is a blank Unity project set up for GameOff 2025, with only the Photon PUN2 asset imported.

## How to Start
1. Open the project in Unity Hub.
2. Make sure Photon PUN2 is installed (check `Assets/Photon` folder).
3. Start building your game!

## Notes
- All custom scripts, scenes, and assets have been removed.
- The project is ready for your team to add new content.
- No compilation errors or safe mode issues should occur.

## For Multiplayer
- Use Photon PUN2 for networking.
- Add your own scripts and scenes as needed.
