## Quick Setup

1. **Install PUN2**: See `PUN2_WEBGL_SETUP.md` for detailed instructions
2. **Get Photon App ID**: Register at https://dashboard.photonengine.com/
3. **Configure Unity**: Add App ID to Photon settings
4. **Build for WebGL**: File > Build Settings > WebGL

## Project Structure

- **Assets/Scripts/NetworkManager.cs** - PUN2 connection & room management
- **Assets/Scripts/PlayerController.cs** - Networked player movement
- **Assets/Scenes/SampleScene.unity** - Main game scene
- **PUN2_WEBGL_SETUP.md** - Complete setup guide
